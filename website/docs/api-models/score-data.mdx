# ScoreData

| 參數名稱                         | 型態                             | 描述                             |
| ------------------------------- | ------------------------------- | ------------------------------- |
| scores | `List<Score>` | 課堂成績列表 |
| detail | `Detail` | 成績細節資料 |

### Score

課堂成績

| 參數名稱                         | 型態                             | 描述                             |
| ------------------------------- | ------------------------------- | ------------------------------- |
| courseNumber | `String` | 課程編號 |
| isPreScore | `bool` | 是否為預覽分數 |
| title | `String` | 課程名稱 |
| units | `String` | 學分數 |
| hours | `String` | 上課時數(單位：小時) |
| required | `String` | 課程類型，選修、必修等等 |
| at | `String` |  |
| middleScore | `String` | 期中成績 |
| generalScore | `String` | 平時成績 |
| finalScore | `String` | 期末成績 |
| semesterScore | `String` | 學期最終成績 |
| remark | `String` | 註記 |

### Detail

學期成績細節

| 參數名稱                         | 型態                             | 描述                             |
| ------------------------------- | ------------------------------- | ------------------------------- |
| creditTaken | `double` | 可獲得學分數 |
| creditEarned | `double` | 實際獲得學分數 |
| conduct | `double` | 操行成績 |
| average | `double` | 平均分數 |
| classRank | `String` | 班級排名 |
| departmentRank | `String` | 系上排名 |
| classPercentage | `double` | 班上排名百分比 |
