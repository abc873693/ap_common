# CloudMessage

提供基本雲端通知所需的基本欄位，有整合 [`Hive`](https://pub.dev/packages/hive) 的 `HiveObject`

| 參數名稱                         | 型態                             | 描述                             |
| ------------------------------- | ------------------------------- | ------------------------------- |
| title | `String` | 標題 |
| dateTime | `DateTime?` | 時間 |
| content | `String?` | 內容 |
| url | `String?` | 網址 |
| imageUrl | `String?` | 圖片網址 |
| data | `Map<String, dynamic>?` | 其他資料 |
