# Announcement

作為首頁輪播所使用的類別，與最新消息審查系統共用類別

## AnnouncementData

| 參數名稱                         | 型態                             | 描述                             |
| ------------------------------- | ------------------------------- | ------------------------------- |
| data | `List<Announcement>` | 最新消息列表 |

## Announcement

| 參數名稱                         | 型態                             | 描述                             |
| ------------------------------- | ------------------------------- | ------------------------------- |
| title | `String` | 標題 |
| id | `String` | 最新消息id |
| tags | `List<String>` | 標籤，目前用來區隔不同學校的最新消息 |
| nextId | `Location` | 下一個最新消息id |
| lastId | `int` | 最後一個最新消息的id |
| weight | `int` | 權重 `HomeScaffold` 會從權重大的排序顯示 |
| imgUrl | `String` | 圖片網址 |
| url | `int` | 活動連結 |
| description | `int` | 描述 會在 `AnnouncementContentPage` 顯示 |
| publishTime | `int` | 發布時間，格式 `YYYY-MM-DDTHH:mm::ss` |
| expireTime | `int` | 過期時間 `HomeScaffold` 會讓超過此時間的最新消息不顯示，格式 `YYYY-MM-DDTHH:mm::ss` |
| applicant | `String` | 申請人名字 |
| applicationId | `String` | 申請人 ID，通常為電子信箱 |
| reviewStatus | `bool` | 審查狀態，若為 `null` 代表尚未審查，`true` 代表通過，`false` 代表失敗 |
| reviewDescription | `String` | 審查說明，由審查人員填寫，讓申請者得知審查不通過原因 |
