# Announcement

作為首頁輪播所使用的類別，與最新消息審查系統共用類別

## AnnouncementData

| 參數名稱 | 型態 | 描述 |
| --- | --- | --- |
| data | `List<Announcement>` | 最新消息列表 |

## Announcement

| 參數名稱 | 型態 | 描述 |
| --- | --- | --- |
| title | `String` | 標題 |
| id | `int?` | 最新消息 id |
| tags | `List<String>?` | 標籤，用來區隔不同學校的最新消息 |
| nextId | `int?` | 下一個最新消息 id |
| lastId | `int?` | 最後一個最新消息的 id |
| weight | `int` | 權重，`HomeScaffold` 會從權重大的排序顯示 |
| imgUrl | `String` | 圖片網址 |
| url | `String?` | 活動連結 |
| description | `String` | 描述，會在 `AnnouncementContentPage` 顯示 |
| publishedTime | `String?` | 發布時間，格式 `YYYY-MM-DDTHH:mm:ssZ` |
| expireTime | `String?` | 過期時間，`HomeScaffold` 會讓超過此時間的最新消息不顯示 |
| applicant | `String?` | 申請人帳號（通常為電子信箱） |
| applicationId | `String?` | 申請 ID |
| reviewStatus | `bool?` | 審查狀態：`null` = 尚未審查、`true` = 通過、`false` = 駁回 |
| reviewDescription | `String?` | 審查說明，由審查人員填寫 |

## AnnouncementLoginData

JWT 登入資料，由後端 `/login`、`/oauth2/google/token`、`/oauth2/apple/token` 回傳。

| 參數名稱 / Getter | 型態 | 描述 |
| --- | --- | --- |
| key | `String` | JWT token |
| decodedToken | `Map<String, dynamic>` | 解碼後的 JWT payload |
| isExpired | `bool` | Token 是否過期 |
| level | `PermissionLevel` | 權限等級 |
| loginType | `String?` | 登入方式（`General` / `Oauth2` / `Apple_sign_in`） |
| username | `String?` | 使用者帳號 |

## PermissionLevel

```dart
enum PermissionLevel { user, editor, admin }
//                      0      1       2
```

| 等級 | 值 | 說明 | 來源 |
| --- | --- | --- | --- |
| user | 0 | 一般使用者：可申請公告、查看自己的申請 | 預設 |
| editor | 1 | 編輯者：可新增/編輯/刪除公告、審核申請、管理黑名單 | 由 Admin 透過 `/auth/editor` API 管理 |
| admin | 2 | 管理員：同 editor + 管理編輯者名單 | 後端環境變數 `ADMIN` 設定 |

## ApiResult\<T\> 與 DataState\<T\>

### ApiResult\<T\>

API 回應的型別安全封裝：

```dart
sealed class ApiResult<T> {}
class ApiSuccess<T> extends ApiResult<T> { final T data; }
class ApiError<T> extends ApiResult<T> { final GeneralResponse response; }
class ApiFailure<T> extends ApiResult<T> { final DioException exception; }
```

### DataState\<T\>

UI 狀態管理用的 sealed class：

```dart
sealed class DataState<T> {}
class DataLoading<T> extends DataState<T> {}
class DataLoaded<T> extends DataState<T> { final T data; final String? hint; }
class DataError<T> extends DataState<T> { final String? hint; }
class DataEmpty<T> extends DataState<T> { final String? hint; }
```

### ApiResult.toDataState()

將 API 回應轉換為 UI 狀態：

```dart
final ApiResult<List<Announcement>> result = await api.getAnnouncements();
final DataState<List<Announcement>> state = result.toDataState();
```
