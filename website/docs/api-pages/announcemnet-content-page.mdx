# 最新消息詳細資訊 AnnouncementContentPage

最新消息詳細資訊，採用頁面顯示。支援直向/橫向自動排版，點擊圖片可進入全螢幕預覽（PhotoView）。

手機介面

 <img src="../../img/pages/announcement_content_page/mobile.jpg" alt="drawing" width="200"/>

平板 or 桌面版介面

 <img src="../../img/pages/announcement_content_page/tablet.png" alt="drawing" width="600"/>


| 參數名稱 | 型態 | 描述 |
| --- | --- | --- |
| announcement | `Announcement` | 最新消息資料（必填） |

## 使用方式

```dart
ApUtils.pushCupertinoStyle(
  context,
  AnnouncementContentPage(announcement: item),
);
```

## 特性

- 標題與描述使用 `colorScheme.onSurface` / `colorScheme.onSurfaceVariant`
- 描述中的連結可點擊（使用 `SelectableLinkify`）
- 外部連結按鈕使用 M3 `FilledButton`
- 點擊圖片進入全螢幕 `PhotoView` 預覽
- 直向模式：圖片在上、文字在下
- 橫向模式：圖片與文字左右並列
