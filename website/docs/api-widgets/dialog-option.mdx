# 選項對話框中選項 DialogOption

:::caution 使用時必須先完成 
 - [主題色設定](../installation/theme)
:::

對話框 `SimpleOptionDialog` 使用的選項，也可自訂 Dialog 使用它

| 參數名稱                         | 型態                             | 描述                             |
| ------------------------------- | ------------------------------- | ------------------------------- |
| text | `String` | 內容文字 |
| check | `bool` | 是否顯示打勾的圖示 |
| onPressed | `Function()` | 點擊事件 |
