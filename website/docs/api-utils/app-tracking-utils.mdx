# AppTrackingUtils

:::caution 注意
使用時必須先完成 [相關設定](../installation/app-tracking-transparency)
:::

提供顯示 App 透明度追蹤的對話框說明

## isSupportAppTrackingApi 

是否支援 `App Tracking API`，目前只僅於 iOS 14 以上會回傳 `true`，其餘平台皆為 `false`

## trackingAuthorizationStatus 

封裝套件 [`App_tracking_transparency`](https://developer.apple.com/documentation/apptrackingtransparency) 的變數`trackingAuthorizationStatus`

## show 

## show 

立即顯示通知

| 參數名稱                         | 型態                             | 描述                             |
| ------------------------------- | ------------------------------- | ------------------------------- |
| context | `BuildContext` | Flutter Widget context |
| onTap | `VoidCallback` | 當為 `null` 時會使用已實作的 `AppTrackingTransparency`，若不為 `null` 則需要自行實作 |
