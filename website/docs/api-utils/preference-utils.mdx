# Preferences 工具

封裝 Flutter 官方維護的 [shared_preferences](https://pub.dev/packages/shared_preferences)，支援所有平台，另外支援使用 [encrypt](https://pub.dev/packages/encrypt) 加密字串(AES cbc 模式)的功能

:::caution 注意
使用時必須先完成 [相關設定](../installation/shared-preferences)
:::

支援 

 - `String`
 - 加密 `String`
 - `int`
 - `double`
 - `String List`

## init

初始化，`ApIcon` 目前的也透過此階段設定

| 參數名稱                         | 型態                             | 描述                             |
| ------------------------------- | ------------------------------- | ------------------------------- |
| key | `Key` | AES 加密演算法中的 key |
| iv | `IV` | AES 加密演算法中的 iv key |
| initialApIcon | `bool` | 是否此階段初始化 ApIcons |
