# 校務通系列 Firebase 共用

基於應用校務通系列使用到原生(Native)的功能製作的Plugin

## 套件使用要求
 - Flutter `v1.20` 以上
 - [ap_common](https://pub.dev/packages/ap_common) `v0.11` 以上

## Getting Started

在 `pubspec.yaml` 中加入 package

v0.1.2 版本以後

```yaml
    ap_common_firebase: ^0.1.2
```


v0.5.0 版本以前

```yaml
  ap_common_firebase:
    git:
      url: https://github.com/abc873693/ap_common_plugin
      ref: v0.1.2
```

執行加入套件

```bash
  $ flutter package get
```
